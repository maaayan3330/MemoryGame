## 🧠 Memory Game – Brooklyn Nine-Nine Edition

A simple interactive Memory Card Game built with React and Vite.
Flip the cards, find the matching characters from Brooklyn Nine-Nine, and try to complete the board using the fewest moves.

## ✨ Features

* Dynamic game board
* Card flip animation
* Matching pair detection
* Move counter
* Automatic win detection
* Game restart option
* Shuffled cards on restart
* Victory screen with final score

## 🛠 Built With

* React

* Vite

* JavaScript (ES6)

* CSS

## 🚀 Live Demo

https://memory-game-two-rosy.vercel.app/

## 📚 Learning Purpose

This project was built as part of my self-learning journey in React.  
The goal was to practice component architecture, state management, and interactive UI development by implementing a complete memory game from scratch.
